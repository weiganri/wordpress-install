Hello, welcome to the github project page of the Wordpress auto-installation script for Ubuntu and Debian. We work hard, to get things done better. But at the top you can download the latest release.

Installation

If you use Ubuntu or Debian there are just a few steps needed to get a full running Wordpress Server just out of the box.

First you have to get the code. So login into putty or directly into your terminal and paste the following commands:

$ cd
$ sudo apt-get install git
$ git clone https://github.com/weiganri/wordpress-install.git
$ cd wordpress-install

Now you will have to start the installation-routine:

$ sudo bash install.sh

Answer all questions with "Yes" and "Enter" and paste in youre favourite password for the root user of the mysql server.

Then open http://youip/phpmyadmin and you'll see a new Database. That's your base for installation of Wordpress.

Go now to http://yourip and type in all database settings you've configured in the setup in your terminal or ssh client. Then go to the next step, copy the code in the text box and get back to terminal.

Now paste the following:

$ sudo nano wp-config.php

Right-klick and paste Klick ctrl + w Klick Y

Go back to your opened browser tab and klick on "let's go". And now have fun with your new Wordpress Blog.
